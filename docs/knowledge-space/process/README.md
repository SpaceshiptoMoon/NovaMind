# 历史与过程文档

本目录保存知识空间重组、迁移和改造过程中的历史材料。

这些文档对理解设计背景、迁移路径和阶段性判断有价值，但它们不是当前仓库状态的唯一事实来源。

## 文档列表

- [`knowledge-reorg-status.md`](./knowledge-reorg-status.md)：知识模块重组完成状态记录
- [`knowledge-reorg-plan.md`](./knowledge-reorg-plan.md)：原始重组计划
- [`knowledge-reorg-migration-summary.md`](./knowledge-reorg-migration-summary.md)：迁移结果和引用更新摘要
- [`IMPROVEMENT-enterprise-kb.md`](./IMPROVEMENT-enterprise-kb.md)：更长期的企业知识库改进草案

## 使用约定

- 需要接手历史工作、追溯设计演进或解释旧决策时，再阅读这里。
- 如果本目录内容与 `../current/` 或当前代码冲突，以 `../current/` 和当前代码为准。
- 某项历史文档中的结论如果仍然有效，应提炼并迁移到 `../current/` 中，而不是继续把过程文档当正式设计文档使用。
