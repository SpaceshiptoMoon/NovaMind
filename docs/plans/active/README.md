# Active Plans

本目录保存仍可能继续演进、仍值得作为后续优化输入的计划文档。

这些文档通常满足至少一个条件：

- 其中的目标还没有完全完成
- 其中的结构清理方向对当前仓库仍有参考价值
- 它们仍适合作为下一轮工程整理的工作底稿

## 使用约定

- 这些文档仍然是计划材料，不应替代正式设计文档。
- 若其中结论已经成为当前事实，应把结论迁移到 `docs/knowledge-space/current/` 或其他正式文档中。
